Jak odpalić aplikację

1.Umieść w folderze z jakąkolwiek nazwą 
2.Wejdź w cmd i wpisz ścieżkę do filderu (Umieść na pulpicie filder po dalsze instrukcje) 
3.Wpisz "cd desktop" po czym wpisz "cd /twoja nazwa folderu/" Wpisz ten kod poniżej g++ main.cpp Inventory.cpp Storage.cpp -o magazyn Gratuluję mam twoje dane

Co aplikacja potrafi?
Umożliwia zapisywania danych o tym co znajduje się w magazynie
Dodawania do już stworzonych danych
Blokady przed dodaniem tego samego Id
